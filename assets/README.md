## How These Are Generated

Chart images and captions are generated by the test scripts in the project root:

```bash
# Generate Twitter and Instagram chart samples
python test_charts.py
```

The output files written to `assets/images/` and `assets/captions/` are committed to the repo so the README renders previews on GitHub without needing a live database connection.

## Usage in README

These files are referenced directly in the root `README.md`:

```markdown
![Twitter Bar Chart](./assets/images/test_twitter_bar_chart_offline.png)
```

## Notes

- These are **sample outputs only** — no real player data is included
- To regenerate them, run the test scripts with your local environment configured
- Do not commit images with sensitive or personal data
